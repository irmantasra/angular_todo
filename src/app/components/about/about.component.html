About angular todo app

This is todo list application written by Irmantas Ramanauskas due to learn some angular stuff.

Angular 6.0.3

Instructions
  • Create new todo - write a title in upper input field and press enter.
  • Change completed status - single click left icon of the todo item.
  • Delete todo - single click right x icon.
  • Edit todo title - single click todo title, write a new one and press enter
